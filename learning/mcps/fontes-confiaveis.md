# MCP: Fontes Confiaveis do Elucy

## Proposito
Lista curada de fontes confiaveis para pesquisa de mercado, estrategia, cenario competitivo e contexto macroeconomico. Quando o Elucy precisar buscar dados externos para enriquecer analises, estrategias ou briefings de Titans, deve priorizar essas fontes.

## Regras de Uso

1. **Prioridade:** Winning by Design e Gartner para RevOps/GTM. McKinsey/Bain/BCG para estrategia enterprise. HBR para argumentos de autoridade em Challenger
2. **Brasil:** Valor + NeoFeed + Brazil Journal para contexto de empresas brasileiras. IBGE/IPEA para dados macro reais
3. **Nunca inventar dados:** Se nao encontrar dado real nestas fontes, declarar `[DADO NAO CONFIRMADO]` — nunca fabricar estatisticas
4. **Business Strategy Generator (Doc 14.1):** Quando gerar deck de estrategia para Titan, usar IBGE + fonte setorial + fonte global como base
5. **Reframes Challenger:** HBR e McKinsey sao as fontes mais fortes para construir provocacoes com base factual
